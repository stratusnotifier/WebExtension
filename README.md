# StratusNotifier
Get alerts when your favorite maps are playing on the Minecraft server Stratus Network.

# Install it on your favorite browser
* Firefox: https://addons.mozilla.org/en-US/firefox/addon/stratus-notifier/
* Chrome: Coming soon
* Opera: Coming soon
